# 🤖 AI Smart Split

AI Smart Split is a smart expense splitter web application built using a single **HTML file** with **inline CSS and JavaScript**.

The project helps users manage and split group expenses easily by calculating balances automatically and minimizing the number of transactions required for settlement.

This lightweight project is simple, fast, and easy to deploy on GitHub Pages without any external frameworks or libraries.

---

# ✨ Features

- ✅ Add participants
- ✅ Add shared expenses
- ✅ Automatic expense calculation
- ✅ Smart balance settlement
- ✅ Transaction minimization
- ✅ Responsive user interface
- ✅ Fully browser-based
- ✅ Single-file project
- ✅ No external dependencies

---

# 🛠️ Built With

- HTML5
- Inline CSS3
- Inline JavaScript
